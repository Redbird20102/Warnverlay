# Warnverlay

Warnverlay is a real-time weather alert overlay designed for live streaming. It displays active weather warnings in a clean, customizable format, making it ideal for streamers, weather broadcasters, and emergency coverage.

---

##  Features

- Real-time weather alerts (using NWWS for faster data and weather.gov api as fallback)
- Visual alert overlays
- 🇺🇸 / 🇨🇦 Support for U.S. and Canadian alerts
- Customizable alert display
- OBS & streaming software compatible
- Optional audio alerts

---

## Installation

### Use Hosted Version
1. Open: https://redbird20102.github.io/Warnverlay/
2. Add it as a **Browser Source** in OBS
3. Adjust width/height to fit your scene
